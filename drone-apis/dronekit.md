# DroneKit

[DroneKit-Python](https://github.com/dronekit/dronekit-python/) allows you to control ArduPilot using the Python programming language.

The [official DroneKit Python documentation](https://dronekit-python.readthedocs.io/en/latest/) contains a [quick start guide](https://dronekit-python.readthedocs.io/en/latest/guide/quick_start.html).
